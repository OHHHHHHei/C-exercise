## 📖 项目介绍 (Introduction)

欢迎来到我的 C 语言练习仓库！

这是一个用于记录我 C 语言学习历程的代码库。主要包含平时课程的作业代码、期中期末考试复习资料，以及在 [洛谷 (Luogu)](https://www.luogu.com.cn/) 上进行的算法题解。

## 📂 目录结构 (Directory Structure)

目前仓库包含以下主要模块：

### 1. 🧩 洛谷刷题 (Luogu Online Judge)
> 目录: `./luogu/`

存放我在洛谷平台上通过的题目代码，文件名对应题目编号。

### 2. 🎓 学校作业 (XMU Homework)
> 目录: `./xmu-homework/`

记录课程布置的日常编程作业，按章节或日期命名。
* 包含指针、结构体、循环控制等专项练习。

### 3. 📝 考试复习 (Exams)
> 目录: `./midterm-exam/`

存放期中/期末考试相关的复习代码和模拟题解。

### 4. 🛠️ 杂项与测试 (Misc)
> 目录: `./杂项/`

* 日常测试的小程序（Hello World, GCD 等）。
* 包含少量 Python 脚本 (`.py`) 用于辅助验证算法或快速计算。

## ⚙️ 开发环境 (Environment)

* **IDE**: Visual Studio Code
* **Compiler**: GCC / MinGW
* **Configuration**: 包含 `.vscode` 配置，开箱即用（`launch.json`, `tasks.json`）。

## 🚀 如何运行 (How to Run)

本项目使用标准的 C 语言编写，可以通过 GCC 编译器运行。
